# Two Pointers
This pattern is useful when doing certain manipulation on linear data structures like arrays and
linked-lists. A variant of it may be applicable to trees, haven't seen a problem like that so far.

# Usual setup
1. Initialize 2 pointers pointing at different location of a linear structure.
2. Maintain specified invariant/property and/or perform the tasks while updating the two pointers.
    - The problem will usually specify a property/invariant that these 2 pointers shall maintain or
      require some tasks like swapping values between them.
3. Advance both pointers: Sometimes towards each other, sometimes same direction.
